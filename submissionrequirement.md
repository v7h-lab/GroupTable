# Yelp AI API Hackathon - Submission Requirements

## Project Requirements
- **Type:** Must be a **NEW** web or mobile application.
- **Core Technology:** Must use the **Yelp AI API endpoint** as the primary data source.
- **Originality:** Must be original work created specifically during the submission period.

## Submission Materials
1.  **Public Code Repository:**
    -   Must contain all source code.
    -   Must include setup instructions.
2.  **Hosted Build / Live App:**
    -   URL to a hosted build (e.g., APK, TestFlight) or a published application for testing.
3.  **Demonstration Video:**
    -   Length: 3 minutes.
    -   Platform: YouTube, Vimeo, or similar.
    -   Content: Must show the app in action.
4.  **Project Description:**
    -   Detailed text explaining features and functionality.
5.  **Yelp AI API Client ID:**
    -   The ID generated during sign-up must be included.

## Technical Criteria
-   The project must thoroughly leverage the Yelp AI API.
-   The application must function as demonstrated in the video.
